# 课程项目摘要与报告提纲

## 1. 课题背景

无人机在动态环境中执行自主导航时，需要同时满足两类目标：

1. 到达目标点并保持较好的路径效率
2. 在存在静态/动态障碍物时避免碰撞

仅依靠经典规划方法往往难以在复杂动态环境中保持实时性；仅依靠强化学习策略又可能在局部状态下输出不安全动作。因此，将“强化学习名义策略”和“安全屏蔽层”结合，是一个比较适合作为课程项目展开的研究方向。

## 2. 方法结构

建议在报告中使用下面的三层结构：

### 2.1 名义策略层

- 作用：根据当前状态和目标位置输出控制动作
- 在高保真版本中，对应 Isaac Sim / OmniDrones 训练环境中的导航策略
- 在轻量级演示中，用一个目标驱动策略近似展示策略输出

### 2.2 安全屏蔽层

- 作用：在动作真正执行前，预测下一步或短时域风险
- 若预测到与动态障碍物的间距低于安全阈值，则对动作做修正
- 本仓库轻量级演示中采用的是基于距离预测的安全屏蔽

### 2.3 动态环境层

- 障碍物按照给定速度运动
- 当障碍物运动到边界时进行反弹
- 用于构造“策略有效但不一定总安全”的测试场景

## 3. 仓库中的两级实验

### 3.1 轻量级仿真

位置：`scripts/run_lite_sim.py`

特点：

- 无需 Isaac Sim
- 可直接生成 GIF
- 适合课程答辩、快速演示与可视化说明安全屏蔽触发过程

### 3.2 高保真 Isaac Sim 训练

位置：`isaac-training/`

特点：

- 面向更真实的无人机训练环境
- 保留了导航环境 `SDFNav`
- 适合说明项目并非只有二维玩具演示，而是有进一步扩展到高保真仿真的基础

## 4. 可写进报告的“本人工作”

为了让课程报告更聚焦你的项目贡献，可以强调以下整理和实现工作：

- 对原始压缩包进行了课程化重构与精简
- 重写项目说明，建立清晰的运行入口
- 补齐缺失的导航任务配置与训练配置
- 修复依赖脚本，使项目具备更好的可运行性
- 补充了一个可直接复现的动态环境安全屏蔽仿真演示

## 5. 评价指标建议

报告中建议至少统计以下指标：

- 成功到达率
- 平均路径长度
- 平均最小安全距离
- 碰撞次数 / 碰撞率
- 安全屏蔽触发次数

## 6. 可直接写进报告的局限性

- 轻量级演示是为了课程展示方便，不等价于高保真物理仿真
- 当前下载包缺失原仓库中的部分演示与部署模块，因此课程提交版聚焦于已有导航训练代码和新增的安全屏蔽演示
- 若进一步做真实无人机实验，还需要补充传感器、控制器和真实平台接口
